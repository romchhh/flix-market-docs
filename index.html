Зміст звіту

FlixMarket - Платформа для підписок на стрімінгові сервіси

FlixMarket - це сучасна веб-платформа для продажу та управління підписками на популярні стрімінгові сервіси. Платформа забезпечує зручний інтерфейс для покупки підписок з підтримкою різних валют, локалізації та безпечних методів оплати через Monobank.

Цільова аудиторія

Користувачі, які бажають придбати підписки на стрімінгові сервіси за вигідними цінами з можливістю оплати в різних валютах та регіонах.

Стек технологій

Платформа для розробки

Next.js 15
React 19
TypeScript
Tailwind CSS

Next.js 15 з React 19 використовується як повноцінний full-stack фреймворк. TypeScript забезпечує типізацію та кращу розробку. Tailwind CSS використовується для швидкого створення адаптивного та сучасного дизайну.

База даних

SQLite
TypeORM

SQLite використовується як легка реляційна база даних для розробки та тестування. TypeORM забезпечує ORM функціональність з підтримкою TypeScript та декораторів для роботи з сутностями.

Система оплати

Monobank API
NextAuth.js
GeoIP

Monobank API інтегровано для безпечної обробки платежів. NextAuth.js забезпечує аутентифікацію користувачів. GeoIP використовується для автоматичного визначення регіону та валюти.

Frontend розробка

Next.js 15 + React 19

Клієнтська частина застосунку розроблена з використанням Next.js 15 та React 19 для створення сучасного, швидкого та SEO-оптимізованого веб-додатку.

Основні особливості:

  • App Router для сучасної маршрутизації та layout системи
  • Server-Side Rendering (SSR) та Static Site Generation (SSG)
  • TypeScript для типізації та кращої розробки
  • Tailwind CSS для швидкого створення адаптивного дизайну
  • Framer Motion для плавних анімацій та переходів
  • Мультимовна підтримка з динамічним перемиканням
  • Автоматичне визначення геолокації користувача
  • Responsive дизайн для всіх типів пристроїв

Backend розробка

Next.js API Routes

Серверна частина реалізована через Next.js API Routes, що забезпечує повноцінний full-stack функціонал в одному фреймворку.

Основні компоненти:

  • API Routes для обробки HTTP запитів
  • NextAuth.js для аутентифікації та авторизації
  • TypeORM для роботи з базою даних
  • Monobank API інтеграція для платежів
  • Webhook обробка для підтвердження платежів
  • Bcrypt для хешування паролів
  • Formidable для завантаження файлів
  • GeoIP для визначення локації користувача
  • Class-validator для валідації даних

База даних

SQLite + TypeORM

Реляційна база даних SQLite з TypeORM використовується для зберігання всіх даних застосунку з підтримкою типізації та декораторів.

Висновки

Результати проєкту

Проєкт FlixMarket було присвячено розробці сучасної веб-платформи для продажу підписок на стрімінгові сервіси. Його реалізація має важливе практичне значення, оскільки забезпечує зручний доступ до популярних сервісів з підтримкою різних валют та регіонів.

Технічна реалізація

Робота над проєктом охопила всі етапи життєвого циклу ПЗ — від аналізу вимог до впровадження повноцінного функціоналу. Застосовано Next.js 15 з React 19, TypeORM для роботи з базою даних, NextAuth.js для аутентифікації та Monobank API для безпечних платежів.

Платіжна система

Інтеграція з Monobank API забезпечує безпечну обробку платежів з підтримкою різних валют (UAH, EUR, USD). Webhook система гарантує надійне підтвердження платежів, а автоматичне визначення геолокації забезпечує правильну локалізацію цін.

Підсумок

Інтерфейс побудовано за допомогою сучасних технологій з підтримкою мультимовності (EN/DE/UA) та адаптивного дизайну. Проєкт став прикладом ефективного використання full-stack Next.js для створення безпечної, масштабованої та зручної e-commerce платформи.
